# Foods Labo

## サイト概要
### サイトテーマ
食材のレビューアプリ

### テーマを選んだ理由


### ターゲットユーザ
- 健康・綺麗になりたい人
- ダイエットをしたい人
- 食材の成分ではなく、本当に効果(実感)のある食材を知りたい人

### 主な利用シーン
- 自分の悩みに効果的な食材が分からない時
- ダイエットに効果的な食材が知りたい時
